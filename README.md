# Capital Consciente - Hub Solar & News Portal

Este é um tema WordPress personalizado, desenvolvido do zero para o nicho de energia solar. Ele combina um portal de notícias de alta performance com um marketplace de instaladores no modelo SaaS.

## 🚀 Principais Funcionalidades

### 1. Arquitetura de Silo (SEO Local)
- Estrutura hierárquica focada em cidades e estados.
- Páginas de taxonomia (`regiao`) otimizadas como Landing Pages.
- Breadcrumbs nativos para sinalização de autoridade no Google.

### 2. Ecossistema SaaS para Instaladores
- **Dashboard Privado**: Painel intuitivo para gestão de perfil corporativo.
- **Integração Stripe**: Sistema de assinaturas recorrentes com suporte a Cartão e Pix.
- **Fair Rotation**: Algoritmo que prioriza membros Premium no topo das buscas com rotatividade justa.

### 3. Prova Social e Conversão
- **Sistema de Reviews**: Avaliações nativas com média de estrelas.
- **JSON-LD Schema**: Microdados de `LocalBusiness` e `AggregateRating` para estrelas nos resultados do Google.
- **Newsletter AJAX**: Captura de leads sem recarregamento de página.

## 🛠️ Tecnologias Utilizadas
- **WP Core**: Plugin-free (máxima performance).
- **Styling**: Tailwind CSS (CDN/Custom).
- **Pagamentos**: Stripe API.
- **SEO**: Silo Architecture & Rich Snippets.

## 📦 Como Instalar
1. Clone este repositório na pasta `wp-content/themes/`.
2. Ative o tema no painel administrativo do WordPress.
3. Configure as chaves do Stripe no arquivo `functions.php`.
